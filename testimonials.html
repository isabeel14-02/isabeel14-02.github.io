HyperText Markup Language

HTML

Función: Es el lenguaje estándar para crear la estructura de las páginas web. Se utiliza para definir el contenido de la web, como encabezados, párrafos, imágenes, enlaces, tablas, formularios, etc.

Cascading Style Sheets

CSS

Función: Se usa para definir el estilo visual y el diseño de una página web. CSS controla aspectos como colores, fuentes, márgenes, disposición de los elementos en la pantalla, etc.

JavaScript:

Script

Función: Es el lenguaje de programación que se utiliza para agregar interactividad a las páginas web. Permite manipular los elementos de la página de manera dinámica, como responder a eventos de usuario (clics, desplazamiento, teclas, etc.).

React

Frameworks/Bibliotecas

Biblioteca de JavaScript para construir interfaces de usuario interactivas.
